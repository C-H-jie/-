# tieba_crawler
此爬虫用于爬取百度贴吧发帖数据，单线程，配备ip池
写于2022.3.31

前排说明：数据不代表最终结果，因为网络波动和程序设计的缺陷（可能有bug），数据不代表最终票数，也没有权威性。仅仅是我个人出于爱好搜集并分析。

统计方式：当有人投票时会自动到选举帖子下回复“XXX为你投上一票！ +1” 我统计了伊藤诚和呱管帖子下这段话出现的次数，用这个次数来当票数。

最终投票统计结果我已经放在了excel文件中。
